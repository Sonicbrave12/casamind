# 🧠 Casamind

Casamind is a simple smart home simulation website that automatically turns lights ON when someone enters the house and OFF when they leave.

This project demonstrates the concept of motion-based home automation using HTML, CSS, and JavaScript.

---

## 🚀 Features

* 💡 Simulated smart lighting system
* 🏠 Detect "entry" and "exit" actions
* ⚡ Instant light ON/OFF respons
